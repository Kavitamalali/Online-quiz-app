# Online Quiz Application

## 📌 Project Description

This is a console-based Online Quiz Application developed using Java, JDBC, and SQLite.
The application allows users to register, log in, create quizzes, add questions, take quizzes, and view their scores.
It focuses on implementing backend logic and database integration.

---

## 🚀 Features

* User Registration & Login
* Create Quiz
* Add Multiple-Choice Questions
* Take Quiz
* Instant Feedback (Correct/Incorrect)
* Score Calculation
* Store Results in Database
* View Past Results

---

## 🛠️ Technologies Used

* Java
* JDBC (Java Database Connectivity)
* SQLite (Database)
* Eclipse IDE

---

## 🗄️ Database Tables

* **users** → Stores user credentials
* **quiz** → Stores quiz titles
* **questions** → Stores quiz questions and options
* **results** → Stores user scores
